# Welcome to the Ovellum demo site

This is a small, hand-written static site generated by `ovellum build` running
in `manual` mode against the Markdown files under `examples/manual-site/content/`.

The point: prove that Ovellum can act as a Jekyll-style site builder for
**purely manual** documentation — no source parsing, no merge engine, just
clean prose pages with a sidebar, a right-hand "On this page" outline,
syntax-highlighted code, and an auto / light / dark theme toggle.

## What's on the menu

- A few prose pages under `content/` that you can edit and re-build.
- A nested `content/guides/` directory to show how the sidebar handles groups.
- Front-matter-driven titles and descriptions.
- A working theme toggle in the top right.

## What to try

1. Edit this file and run `pnpm -w run demo:site` again — the changes
   appear in `examples/manual-site/dist/`.
2. Open `dist/index.html` directly, or serve the folder:

```bash
npx serve examples/manual-site/dist
```

3. Click the theme toggle (top right) to cycle through auto → light → dark.
4. Hover any code block — the **Copy** button appears in the upper right.
