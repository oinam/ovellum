# 404（ページが見つかりません）

アクセスしようとしたページは、このサイトには存在しないか、移動されました。

役立ちそうな行き先をいくつか挙げます:

- [ホームページ](/ja/)には、すべてのセクションが一覧されています。
- [はじめに](/ja/docs/)は、ゼロからビルド済みの `dist/` までを3ステップでたどる道のりです。
- [リファレンス](/ja/docs/reference/config/)では、すべての設定フィールドと CLI
  フラグを解説しています。
- このページに外部リンクからたどり着き、そのリンクが本来は機能すべきものであれば、
  [issue を立ててください](https://github.com/oinam/ovellum/issues)。リダイレクトを
  修正するか、ページを復元します。

GitHub Pages は、パスがビルド済みファイルに一致しないときに、このファイルを自動的に
配信します。そのため、独自のルーティングを用意しなくても、実際にページが見つからない
ケースでこのページが表示されます。
